◆ HandlerErrorCode

enum server::handlers::HandlerErrorCode
strong

Enumeration that defines protocol-agnostic hander error condition codes, used by server::handlers::CustomHandlerException.

A handler for a specific protocol (e.g. http) should define mapping from the HandlerErrorCode to protocol-specific error code.

Note
When adding enumerators here, please add a description string to the cpp file
